# Linux Shell Scripting Practice
Basically Shell Scripting is a process of automating our day-to-day activities or regular activities on our linux computer.
A collection of Bash shell scripting practice scripts for beginners and intermediate learners.  
Organized to help you understand basic commands, system monitoring, automation scripts, and filtering techniques.

## Folder Structure
linux-shell-scripts/
├── basics/ # Basic scripts to practice echo, variables, loops.
│ ├── first-shell-script.sh
│ ├── second-shell-script.sh
│ └── test.sh
├── system-scripts/ # Scripts for system monitoring and automation
│ └── node-health-script.sh
└── README.md # This file

##  Features

- Learn and practice **Bash basics**: echo, variables, loops, conditionals  
- System monitoring scripts: disk usage, memory usage, CPU info  
- Data filtering practice using `grep`, `awk`, `sed`  
- Well-commented scripts for easy understanding  
- Organized folder structure for readability
